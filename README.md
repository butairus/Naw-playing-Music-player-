このフォルダを解凍したら、中に"Nawplaying.exe"が入っているのでこれを起動。</br>
左上のフォルダアイコンのボタンから音楽ファイルの入ったフォルダを選びます。</br>
フォルダを選択すると、勝手に曲が流れ始めます。</br>
rndを使ってるので、よく同じ曲が選ばれます..</br>
-----各種曲操作ボタンの説明------</br>
左下から順に</br>
・ミュートボタン（音量を0にします）</br>
・音量バー（バーが右に行くほど音量が大きくなります）</br>
・再生ボタン（曲が一時停止されているときに押すと、曲が一時停止されたところから流れ始めます）</br>
・一時停止ボタン（曲が流れているときに押すと、曲を一時停止します。）</br>
</br>
